Projects

  • Itinerary Optimization With Machine Learning. The goal of this project was to optimize travel routes for a delivery vehicle by using machine learning model predictions. It was a two-component problem: first, I trained a machine learning model on the data to predict how long it would take a delivery vehicle to go from point one point to another, and then I fed these predictions into a genetic algorithm which decided which is the most time efficient visit order for a given set of points. Trip durations were predicted by a gradient boosted regression model, and I set up a Spark cluster on Google Compute Engine Instance to process 450M records. Code and presentation can be found here.

  • Predicting Item Purchases for Instacart. In this project, I forecast whether a customer would reoder a certain product. Essentially, this was classification problem with two classes: 1 if a user reordered a product and 0 otherwise. The data set was taken from Kaggle; I engineered target variable, trained and tuned a logistic regression model with special attention to class imbalance in the data. AWS EC2 was used to process and model on 32M records. Code and presentation can be found here.

  • Predicting Item Purchases for Instacart. For this project, I scraped IMDb with Selenium and BeautifulSoup and analyzed movie trends, making predictions on the IMDb rating based on different sets of features. I trained and cross-validated a model, exploring Lasso and Ridge techniques for regularization. Code and presentation can be found here.

  • DotA 2 Player Analytics. For this project, I chose popular online multiplayer game DotA 2. I analyzed a player's account and looked for insights: gameplay patterns, item choices, and various in-game metrics. Anonymized data was collected using DotA developer API. Code and presentation available here.

  • Income Classification in the US. The goal of this project is to train and tune a binary random forest classifier to predict whether a person makes below or above the 50k threshold and analyze what features are correlated with income. I trained a random forest binary classifier to predict income bracket, performed grid search and randomized search to optimize model parameters. Code can be found here.
